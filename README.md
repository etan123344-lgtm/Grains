Mobile granular sampler for iOS. Record/upload audio, mangle it with the granular engine, and then mix it with the resonator to create something entirely new! 

Granular engine based off the Bastl microGranny source code (https://github.com/bastl-instruments/bastlMicroGranny.git) with some tweaks: 
- allow multiple grains to play
- changes to the randomizing function
- reverse grain playback
